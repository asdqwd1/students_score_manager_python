# students_score_manager_python
## 这是一个用python3.8编写的学生成绩管理系统，具体满足的需求如下：
1.录入学生信息  
2.查找学生信息  
3.删除学生信息  
4.修改学生信息  
5.排序  
6.统计学生总人数  
7.显示所有学生信息  
0.退出系统  

## 注意：
1.程序内只包括英语成绩，python成绩，java成绩的选项
